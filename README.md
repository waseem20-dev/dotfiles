## 🎨 التنسيق والثيم (Theme)
تم اعتماد ثيم **Catppuccin Mocha** بالكامل للحصول على مظهر متناسق واحترافي في كافة أرجاء النظام (Hyprland, Waybar, Fish, VS Code).

## 🖋️ الخطوط المستخدمة (Required Fonts)
لضمان عمل الواجهات والأيقونات بشكل صحيح، يرجى التأكد من تثبيت الخطوط التالية:
- **Caskaydia Cove Nerd Font**: الخط الأساسي للـ Terminal (Kitty) والـ Wofi والـ VS Code Terminal.
- **JetBrainsMono Nerd Font**: يستخدم في شاشة القفل (Hyprlock)، التنبيهات (SwayNC)، والـ Editor.
- **MesloLGS Nerd Font Mono**: الخط المخصص لتنسيق الـ Waybar.
- **Symbols Nerd Font**: ضروري جداً لعرض الأيقونات والرموز في النظام.
- **Adwaita Sans**: خط النظام الافتراضي لواجهات الـ GTK.

## 🛠️ تفاصيل النظام (The Setup)
- **نظام التشغيل:** [CachyOS](https://cachyos.org/) (Arch-based)
- **مدير النوافذ:** Hyprland
- **الصدفة (Shell):** Fish Shell 🐟
- **الجهاز:** Lenovo IdeaPad Gaming 3 (Ryzen 7 + RTX 4050)

## اختصارات النظام (Shortcuts)
لفتح الطرفيه : SUPER + Q
لفتح مدير الملفات : SUPER + E
لفتح المتصفح Brave : SUPER + B
الخروج النهائي من Hyprland (Logout) : SUPER + M
قفل الشاشة فوراً باستخدام Hyprlock : SUPER + L
لفتح wofi : SUPER + R
التقاط صورة للشاشة (Screenshot) : PRINTSCREEN
** هذه بعض الاختصارات وليست جميعها **
